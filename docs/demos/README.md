## Comment enregistrer

- **Captures (PNG)** : `Win + Maj + S` (Windows) / `Cmd + Maj + 4` (macOS).
- **GIF** : [ScreenToGif](https://www.screentogif.com/) (Windows), [Kap](https://getkap.co/) (macOS),
  ou [Peek](https://github.com/phw/peek) (Linux).
- Largeur conseillée : **1280 px** ; gardez les GIF sous ~8 Mo pour un affichage fluide sur GitHub.

> Astuce : lancez l'app en local (`docker compose up -d`) sur `http://localhost:8080`,
> créez un compte de démo et un personnage exemple avant d'enregistrer.
